Day 3: Hands-on Labs

L1: Differential Expression Analysis & Interpretation of DE Results

Lab Instructions: L1

In this lab, you will perform differential expression analysis using DESeq2, explore DE results and outputs, and visualize results with PCA, MA plots, and volcano plots.

Steps:
  1. Run differential expression using deseq2
  2. Explore DE results and outputs
  3. Exploring PCA, MA plots, and volcano plots to assess results and sample structure

L2: Visualization & Integrated QC

Lab Instructions: L2

This lab focuses on visualization and integrated QC. You will generate heatmaps, boxplots, and other summary plots using seaborn in Python, and run MultiQC to aggregate reports.

Steps:
  1. Generate heatmaps, boxplots, and other summary plots (using seaborn in Python).
  2. Run MultiQC to aggregate reports across pipeline steps.

L3: Enrichment Analysis in Practice

Lab Instructions: L3

In this lab, you will perform pathway and enrichment analysis using tools like ClusterProfiler, taking a list of differentially expressed genes and running GO/pathway enrichment.

Steps:
  1. Take the list of differentially expressed genes and run GO/pathway enrichment.
  2. Interpret enriched terms and link them back to the biological question.
