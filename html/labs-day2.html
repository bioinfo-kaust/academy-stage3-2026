Day 2: Hands-on Labs

L1: Quality Evaluation Hands-on

Lab Instructions: L1

This lab covers quality evaluation using tools like FastQC and fastp. You will run these tools on downloaded FASTQ files, interpret quality reports, and identify potential issues.

Steps:
  1. Tools: FastQC, fastp (short reads), NanoPlot (long reads)
  2. Run FastQC on downloaded FASTQ files
  3. Run fastp for quick quality overview
  4. Interpret quality reports and identify potential issues

L2: Quality Control Hands-on

Lab Instructions: L2

This lab focuses on quality control, including trimming and adapter removal. You will use tools like TrimGalore, Cutadapt, fastp, and Porechop, apply trimming and adapter removal, and run post-processing QC.

Steps:
  1. Tools: TrimGalore, Cutadapt, fastp, Porechop
  2. Identifying low-quality and outlier samples
  3. Apply trimming and adapter removal
  4. Run post-processing QC to verify improvements and compare before/after quality metrics

L3: RNA-seq Alignment Hands-on

Lab Instructions: L3

In this lab, you will perform RNA-seq alignment to a reference genome using tools like STAR and samtools, and inspect alignment outputs (BAM files).

Steps:
  1. Tools: STAR, samtools, IGV
  2. Align RNA-seq reads to reference genome
  3. Inspecting alignment outputs (BAM files) and mapping statistics
  4. Make informed post-processing decisions

L4: Counting & Expression Matrix Construction

Lab Instructions: L4

This lab covers read counting and expression matrix construction using tools like Salmon or StringTie. You will perform raw gene counting and assemble and inspect the expression matrix.

Steps:
  • Perform raw gene counting (e.g. featureCounts or similar).
  • Run transcript-level quantification with tools such as Salmon or StringTie.
  • Assemble and inspect the expression matrix; perform basic sanity checks
