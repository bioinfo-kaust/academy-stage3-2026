Program Learning Outcomes

By the end of this program, you will be able to:

📚
Foundational Skills

Develop foundational skills in bioinformatics and genomics data analysis

💻
Linux & Tools

Gain hands-on experience using Linux-based computational environments and standard bioinformatics tools

🧬
Transcriptomics

Build a solid understanding of transcriptomics and RNA sequencing (RNA-seq) principles and applications

📄
Data Formats

Work confidently with commonly used genomics and transcriptomics data formats

🔬
End-to-End Workflow

Experience a complete RNA-seq analysis workflow, from raw data to biological insight

🎓
Problem Solving

Solve real bioinformatics challenges and present bioinformatics findings

Program Overview

Day 1

Foundations & Transcriptomics

Bioinformatics foundations, computing environments, genomic file formats, and introduction to transcriptomics and RNA-seq experimental design.

View Day 1 Schedule →
Day 2

Data Preprocessing Pipeline

Quality evaluation and control, read alignment to reference genome, and read counting/quantification to build expression matrices.

View Day 2 Schedule →
Day 3

Downstream Analysis

Differential expression analysis, visualization and reporting, pathway and enrichment analysis, and extended RNA-seq applications.

View Day 3 Schedule →
Day 4

Presentations & Evaluation

Emerging domains (single-cell, spatial transcriptomics), workflow summary, examination, and group project presentations.

View Day 4 Schedule →
