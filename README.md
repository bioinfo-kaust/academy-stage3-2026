- **Day 1:** Bioinformatics foundations, genomic file formats, transcriptomics introduction
- **Day 2:** Quality control, read alignment (STAR), expression quantification
- **Day 3:** Differential expression (DESeq2), visualization, pathway enrichment
- **Day 4:** Review, group presentations and evaluation
