Bioinformatics Specialization Program – 2026 

 

Program Learning Outcomes 

Develop foundational skills in bioinformatics and genomics data analysis 

Gain hands-on experience using Linux-based computational environments and standard bioinformatics tools

Experience a complete, end-to-end RNA-seq analysis workflow, from raw sequencing data to biological insight, including data processing, read alignment, differential expression, and functional enrichment analysis 

Solve real bioinformatics challenges and present bioinformatics findings

Program Outline 

Day 1: Foundations of Bioinformatics and Transcriptomics 

Objectives: 

Recap of bioinformatics foundations from stage 2 

Get hands-on with bioinformatics computing environment 

Become familiar with genomic data formats and accessing public sequencing datasets 

Introduce transcriptomics concepts, experimental design, and an end‑to‑end RNA‑seq workflow 

Form teams and select applied transcriptomics project topics 

S1: Foundations in Bioinformatics 

Bioinformatics introduction recap 

Applications of bioinformatics 

Sequencing technologies  

Genomics data formats (recap) 

Computing environments and skills for bioinformatics 

L1: Warm-up for large-scale bioinformatics analysis 

Familiarization with the training environment 

Access public genomics data resources 

Download example RNA‑seq datasets from public repositories 

 

S2: Transcriptomics Applications and Analysis 

Biological motivation for transcriptomics and gene expression profiling 

Transcriptomics experimental design and protocols (replicates, depth, batch effects) 

Overview of an end-to-end RNA-seq analysis workflow: from raw reads to insights 

L2: Get hands-on with transcriptomics data 

Execute bioinformatics tools and explore sequencing data 

 

S3: Team Formation and Overview of Applied Projects 

Present applied RNA‑seq project scenarios (e.g. case–control, time‑series). 

Discuss expectations for group work, deliverables, and final presentations. 

L3: Project and Dataset Selection 

Form teams and pick a project topic. 

Search for and download relevant public datasets and associated metadata. 

 

Day 2: The Genomics Data Analysis Pipeline 

Objectives: 

Learn how to evaluate and process raw sequencing data 

Understand key steps in the RNA‑seq pre‑processing pipeline from reads to counts 

Gain hands‑on experience running quality checks, trimming, alignment, and quantification 

 

S1: Quality Evaluation 

Sequencing quality metrics and common RNA-seq artifacts 

Understanding Phred quality scores 

Sources of technical variation in RNA-seq data 

Impact of data quality on downstream analysis 

L1: Quality Evaluation (45 min) 

Tools: FastQC, fastp (short reads), NanoPlot (long reads) 

Run FastQC on downloaded FASTQ files 

Run fastp for quick quality overview 

Interpret quality reports and identify potential issues 

 

S2: Quality Control 

Principles of RNA-seq quality control and preprocessing 

Read filtering and trimming: when and why 

L2: Quality Control (45 min) 

Tools: TrimGalore, Cutadapt, fastp, Porechop 

Identifying low-quality and outlier samples 

Apply trimming and adapter removal 

Run post-processing QC to verify improvements and compare before/after quality metrics 

 

S3: Alignment to Reference Genome 

Reference genomes and gene annotation: versions, builds, and sources 

Brief note: building your own genome resource, when and how (optional) 

Concepts of splice-aware read alignment and multimapping reads 

Alignment quality metrics and typical pitfalls 

L3: Alignment (45 min) 

Toos: STAR, samtools, IGV 

Align RNA-seq rads to reference genome 

Inspecting alignment outputs (BAM files) and mapping statistics 

Make informed post-processing decisions 

 

S4: Read Counting and Quantification 

Gene‑level versus transcript‑level quantification and when each is appropriate. 

Building an expression matrix for multiple samples 

Other examples (brief note): Variant allele counting (similar concept but read counting is at the variant level) 

L4: Counting and Expression Matrix 

Perform raw gene counting (e.g. featureCounts or similar). 

Run transcript‑level quantification with tools such as Salmon or StringTie. 

Assemble and inspect the expression matrix; perform basic sanity checks 

 

 

Day 3: Downstream Analysis, Visualization, and Interpretation 

Objectives: 

Get hands-on experience with downstream processing of gene expression data; from gene counts to biological insights. 

Learn core concepts in normalization, differential expression, and result visualization 

Perform gene set, pathway, and enrichment analysis for biological interpretation 

 

S1: Differential expression analysis 

Concepts of normalization, dispersion in RNA‑seq count tables. 

Statistical methods for significance testing 

Experimental design formula and contrasts for differential expression 

Typical visualizations of differential expression results (PCA, MA plot, volcano plot) 

L1: Perform DE Analysis 

Run differential expression using deseq2 

Explore DE results and outputs 

Exploring PCA, MA plots, and volcano plots to assess results and sample structure 

 

S2: Visualization and Reporting of DE Results 

Principles of effective visualization for transcriptomics results. 

Common plot types: heatmaps, boxplots, sample clustering, and summary plots 

L2: Visualization and Integrated QC 

Generate heatmaps, boxplots, and other summary plots (using seaborn in Python). 

Run MultiQC to aggregate reports across pipeline steps. 

 

S3: Pathway and enrichment analysis 

Concepts of gene ontology (GO), KEGG, and other pathway databases. 

Over‑representation analysis and gene set enrichment analysis for RNA‑seq 

L3: Pathway and enrichment analysis 

Take the list of differentially expressed genes and run GO/pathway enrichment. 

Interpret enriched terms and link them back to the biological question. 

 

S4: Additional applications of transcriptomics  

Discovery of novel transcripts and genes 

Detection of alternative splicing and transcript isoforms. 

Gene fusion detection 

Detection of sequence variants in RNA-seq data

Evaluate participant progress and collect feedback for future iterations 

 

S1: Other applications of RNA-seq 

Single cell transcriptomics

S2: Summary and Future Directions

Brief outlook on other genomic applications (e.g. DNA variant calling, epigenomics
