Group Projects

Each group (4-5 students) will work on an RNA-seq analysis project from raw data to biological insights. Projects use small, publicly available datasets that can be analyzed on your laptop.

Project Deliverables
  • Complete RNA-seq analysis pipeline (QC → Alignment → Quantification → DE → Enrichment)
  • 5-minute flash talk presentation on Day 4
  • Key visualizations: PCA, volcano plot, heatmap, enrichment results
  • Biological interpretation connecting results to the research question

How to Download Data

Important: Download only the samples listed for your project. Each project uses a subset of a larger study to keep file sizes manageable. 
# Activate your environment
conda activate bioinformatics_course

# Download using SRA Toolkit (replace SRR_ID with actual accession)
prefetch SRR_ID
fasterq-dump --split-files SRR_ID
gzip *.fastq

# Or download directly from ENA (often faster)
wget ftp://ftp.sra.ebi.ac.uk/vol1/fastq/SRRXXX/SRRXXXXXXX/SRRXXXXXXX_1.fastq.gz
wget ftp://ftp.sra.ebi.ac.uk/vol1/fastq/SRRXXX/SRRXXXXXXX/SRRXXXXXXX_2.fastq.gz

Project Timeline

Day Project Milestone
Day 1 (PM) Form groups, select project, download data
Day 2 QC, trimming, alignment, quantification
Day 3 Differential expression, visualization, enrichment analysis
Day 4 (AM) Prepare presentation, finalize figures
Day 4 (PM) Flash talk presentations (5 min per group)

Presentation Guidelines

Flash Talk Format (5 minutes)
  1. Background (30 sec): What is the biological question?
  2. Methods (30 sec): Brief pipeline overview
  3. Results (3 min): Key figures - QC summary, PCA, volcano plot, top DE genes, enrichment
  4. Conclusions (1 min): Biological interpretation and main findings
